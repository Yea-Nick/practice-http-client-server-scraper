# HTTP клиент и сервер для сбора информации из интернета

## Возможности
- Сервер хранит соответствие keyword -> urls
- Клиент отправляет keyword, получает список URL
- Клиент выбирает URL и скачивает контент через сервер
- Сервер отдаёт прогресс скачивания (SSE): total/downloaded/percent
- Клиент сохраняет скачанное в LocalStorage и умеет показывать offline

## Локальный запуск

### 1) Сервер
```bash
cd server
npm i
npm start
```

### 2) Клиент
Открыть файл в браузере
```bash
client/index.html
```
